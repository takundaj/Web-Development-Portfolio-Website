POKEDEX

Pokémon are fictional creatures from a Japanese cartoon that was first released in April 1997. This site acts as a Pokémon dictionary allowing the user to view the stats and abilities of each of the first original 150 Pokémon from the series. The site also allows the user to search a Pokémon by name.

Each Pokémon’s information is drawn from an API known as 'PokéAPI', to be displayed.

FLAPPY BIRD CLONE

This is a remake of the famous flappy bird game that took the world by storm in 2013.

The aim of the game is to fly the object through the gaps in the incoming columns without hitting them.

The user’s score is calculated with every game and their high score is updated when surpassed. I also coded the high score to persist, using local storage.

TO-DO LIST

This is a To-Do list application that allows the user to create, delete, and mark the progress of tasks. A progress bar in the header of the application shows the overall amount of tasks that have been completed by the user.

The application also uses local storage so that the user's to-do items persist/remain even when the user restarts the browser.
